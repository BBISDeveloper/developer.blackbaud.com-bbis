Country And State API Endpoints

GET Country

Returns a list of all active countries alphabetized by their description.  Will return information about all active countries alphabetized by their description.

Possible Responses

Status
200 OK
Body
Array of Countries
Description
Returns an array country objects

Example Responses

Success - 200 OK

[{Id:'2C3F57A7-65DE-4D55-89EF-7857E6EDD53C', Abbreviation='CA', Description:'Canada'},{Id:'B4837D3E-36F6-42F4-8049-CC2A2B108F9E', Abbreviation='US', Description:'United States'}]

GET Country/{Id}

Returns information about a specific country.  If the provided country Id is valid, then this will return information about the country.

Possible Responses

Status
404 Not Found
Description
No country was found by the specified Id.
Status
200 OK
Body
Country
Description
Returns a country object

Example Responses

Success - 200 OK

“{Id:'B4837D3E-36F6-42F4-8049-CC2A2B108F9E', Abbreviation='US', Description:'United States'}”

GET Country/{Id}/State

Returns a list of states associated with a country, alphabetized by their description.  If the provided country Id is valid, then this will return a list of state objects for each state associated with the country. The states will be returned alphabetized description.

Possible Responses

Status
404 Not Found
Description
No country was found by the specified Id.
Status
200 OK
Body
Array of States
Description
Returns an array of state objects for the given country alphabetized by their description.

Example Responses

Success - 200 OK

“[{Id:'70BF9733-5D24-46F4-A40C-61F6D2765575', Abbreviation='SC', Description:'South Carolina'}]”

GET Country/{Id}/AddressCaptions

Returns the captions for Address Lines, City, State, and Postcode.  If the provided country Id is valid, then this will return the captions for address lines, city, state, and postcode.

Possible Responses

Status
404 Not Found
Description
No country was found by the specified Id.
Status
200 OK
Body
Address Captions
Description
Returns an AddressCaptions object containing the labels for Address Lines, City, State, and Postcode.

Example Responses

Success - 200 OK

“{AddressLines='Address', City='City', PostCode='Zip', State='State'}”
